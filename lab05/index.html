Lab 1 & 2

Published on Februrary 12, 2025 by Khrystian Green-Dorsey

Over the course of these first 2 labs, we covered the basics of HTML and structuring our webpages. I was already familiar with HTML before completing assignments in this class, but it was enjoyable to refamiliarize myself with all the basics. In fact, I didn't remember some tags like the article or description list tags, so this was a really good refresher. Overall, it was enjoyable to make these first few basic pages.

What I Learned

  1. Basic HTML tags and attributes
  2. How to upload my pages to Github
  3. How to format my code so that it's easily read

These resources were really helpful:

  • W3Schools
  • Stack Overflow
  • MDN Web Docs

Lab 3

Published on Februrary 19, 2025 by Khrystian Green-Dorsey

During this lab, we learned a lot about the basics of using CSS to style our web pages. Even though I knew about CSS already, this was a good refresher because I had never given it much thought before. The most valuable part of this lab, in my opinion, was covering the methods of applying CSS and the benefits of each. Overall, this was fun and gave me a chance to learn more CSS properties to use in my future web pages.

What I Learned

  1. Structure of CSS rules
  2. Different methods of applying CSS to HTML
  3. Styling specific sections/elements with IDs and classes

These resources were really helpful:

  • W3Schools
  • Stack Overflow
  • CSS Subreddit

Lab 5

Published on March 14, 2025 by Khrystian Green-Dorsey

For lab 5, we learned the basics of the box model and more advanced website layouts. Both flex and grid are very helpful for making more interesting layouts for pages, but can be annoying to work with at times. The most valuable part of this project was learning more about media queries, because I haven't used them before. Overall, this project was a good opportunity to practice creating more appealing page layouts.

What I Learned

  1. How to use flexbox & grid for advanced layouts
  2. How to use media queries for more responsive pages
  3. How to use the Box Model

These resources were really helpful:

  • W3Schools
  • Stack Overflow
  • CSS Subreddit
