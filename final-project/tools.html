Tools & Resources

Here, I'll share some of the resources I found most helpful during the semester. These include websites, tools, and fourms that I used to enhance my projects and learning experience.

Documentation & Tutorials

W3Schools
One of the best sites with detailed information on every part of HTML and CSS. Lots of examples of things in use and in multiple ways.
MDN Web Docs
Another of the best sites for detailed information and it includes plenty of activities for practicing.
The Odin Project
A lot of the web dev knowledge I had before this semester comes from this site. They have very detailed lessons to follow and interesting projects that cover every part of web dev (html, css, js, ruby, etc.).

Forums & Chats

Stack Overflow
Essentially a more consistently professional version of reddit. The answers here are usually a lot more technical and detailed than Reddit's answers.
Reddit
Reddit has plenty of subreddits with (some) useful people, lots of good answers to questions you might have. Sometimes I end a web search with "reddit" and get the perfect answer.
ChatGPT
ChatGPT is helpful for plenty of things if you get stuck on something really specific, but more often it'll generate something terrible that you'll be inspired to change on your own.

Tools

W3c Validation Services: HTML, CSS
W3c validators to check that your code is up to date and free of any major errors.
CSS Grid Generator
An interactive tool for creating grid layouts. It's very straightforward and works well.
Text: Cooltext & Dafont
Two great resources for text/logo design. Cooltext is focused on more 2000s/blinkies style text logos and dafont is a collection of various fonts for normal text.
Website Color Design: Colormind & Paletton
Colormind and Paletton are both useful specifically for seeing how a color pallete works on a website. I prefer other sites for actually developing the color palettes.
Color Palettes: Coolors & Color Designer
Both of these sites are really helpful for creating color palettes. Color designer also has really useful tools relating to the color palettes you make, like combining colors or adding colors between ones you have to make a gradient.
NeoCities
NeoCities is a free web hosting service that allows you to create and host your own website. It's one of a few good communities built around creating personal websites, especially ones in older 2000s styles.
Cursors 4 U
A site with a large collection of free cursors, complete with code that can be copy-pasted into your own page.
