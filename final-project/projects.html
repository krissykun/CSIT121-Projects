Labs

Here are the labs I've completed throughout the semester. Each lab showcases different skills and concepts I've learned in this course.


Just click or tap on a tile to view the full lab.

The layouts and parts of color palettes of some sites are the same as normal.

The top of each page for every lab has a navigation strip that leads back to the portfolio pages as well as the original site.


Lab 1: Web Publishing

A very simple about page, to practice uploading pages to Github.

Lab 2: HTML Basics

A basic blog, designed for practicing with HTML's structure tags.

Lab 3: CSS Basics

An expanded version of the blog from lab 2, including CSS styling.

Lab 4: Visual Media

A 4-page website designed to showcase popular online image formats.

Lab 5: Layout Lab

An expanded version of the blog in labs 2 & 3, updated with a CSS flexbox and/or grid layout

Lab 6: UI/UX Evaluation

An evaluation of the accessibility of an existing page, mine is on MED-El's website.

Lab 7: Tables & Forms

Applying table and form knowledge in a fun way with a tier list and contact form.

Lab 8: Interactive Menu

The media showcase website updated with a responsive navigation bar using media queries.

Lab 9: XML

A simple application of XML to create an employee database connected to a button that shows it on screen.

Lab 10: Standards Presentation

A 4-page website designed designed to present on a W3c web standard, mine is on Payment Method Identifiiers.
